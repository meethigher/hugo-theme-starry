# Hugo Theme Starry

Starry Nights. Calm Reading.

## Features

* Card-style layout
* Responsive design
* Starry particle effect
* Light/Dark mode toggle
* Table of contents
* Lazy image loading
* Image viewer
* MathJax lazy rendering
* IndexedDB fast search
* Gitalk comments integration
* Open markdown in browser (dev mode)
